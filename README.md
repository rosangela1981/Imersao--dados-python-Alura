📊 Dashboard de Salários na Área de Dados
Este projeto é uma aplicação interativa desenvolvida em Python utilizando a biblioteca Streamlit. O objetivo é proporcionar uma análise visual e dinâmica sobre a evolução e distribuição dos salários no setor de dados globalmente.

🚀 Tecnologias Utilizadas
Para a construção deste projeto, foram utilizadas as seguintes ferramentas:

Python: Linguagem principal do projeto.

Streamlit: Framework utilizado para transformar scripts Python em aplicações web interativas de forma rápida e eficiente.

Pandas: Biblioteca essencial para manipulação e tratamento dos dados.

Plotly Express: Utilizada para a criação de gráficos dinâmicos e interativos (barras, histogramas, pizza e mapas coropléticos).

Exibição imediata de indicadores-chave baseados nos filtros aplicados:

Média salarial anual (USD).

Total de registros analisados.

Cargo com maior frequência na base de dados.

📊 Análises Visuais
Top 10 Cargos: Gráfico de barras horizontais com os maiores salários médios por função.

Distribuição Salarial: Histograma para entender a concentração de faixas de renda.

Proporção de Trabalho: Gráfico de rosca mostrando a divisão entre trabalho remoto, híbrido e presencial.

Mapa Global: Mapa coroplético focado em Data Scientists, mostrando a média salarial por país de residência.
